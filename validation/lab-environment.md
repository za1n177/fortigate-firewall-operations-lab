# FortiGate Lab Environment

## Platform
- FortiGate VM (FortiOS)
- Local virtualization (VirtualBox / VMware)

## Network Topology
- WAN interface connected via NAT
- LAN interface serving internal subnet

## IP Addressing
- FortiGate LAN: 192.168.10.1/24
- Client VM: 192.168.10.10/24

## Purpose
This lab simulates a real-world enterprise firewall deployment used to validate:
- Traffic flow
- Policy enforcement
- NAT behavior
- Logging and troubleshooting
